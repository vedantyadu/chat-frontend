## Getting started
### Installing dependencies
```bash
npm install
```
### Changing backend URLs
- Change the socket server URL to your socket server URL in `/src/socket/socket.jsx`
- Change the base Axios URL to your server URL in `/src/utils/axiosconfig.js`
